# SportClub React

Este proyecto es una aplicación web para la gestión de un club deportivo, desarrollada con React para el frontend y Node.js para el backend.

## Tecnologías Utilizadas

*   **Frontend:** React, Vite, JavaScript, HTML, CSS
*   **Backend:** Node.js, Express.js..

## Instalación de Dependencias

Para instalar las dependencias de este proyecto, sigue los siguientes pasos:

### Frontend

Navega al directorio `frontend` y ejecuta el siguiente comando:

```bash
npm install
```

### Backend

Navega al directorio `backend` y ejecuta el siguiente comando:

```bash
npm install
```

## Ejecución del Proyecto

Para ejecutar el frontend y el backend, sigue los siguientes pasos:

### Frontend

Navega al directorio `frontend` y ejecuta el siguiente comando:

```bash
npm run dev
```

Esto iniciará el servidor de desarrollo del frontend, generalmente accesible en `http://localhost:5173`.

### Backend

Navega al directorio `backend` y ejecuta el siguiente comando:

```bash
npm start
```

Esto iniciará el servidor del backend, generalmente accesible en `http://localhost:3000`.
